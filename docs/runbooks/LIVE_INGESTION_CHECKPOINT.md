# Live Ingestion Checkpoint

Live ingestion is not enabled in Stage 4A.

Before any controlled live request, the user must explicitly approve:

- W2-specific API key injection method
- one controlled network request
- fixture or competition scope for the test
- expected logging and rollback procedure
- quota policy values for the environment

Final Stage 4A state:

```text
LIVE_INGESTION_CHECKPOINT_REQUIRED
```
