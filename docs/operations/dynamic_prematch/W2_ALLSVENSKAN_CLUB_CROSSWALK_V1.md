# W2 Allsvenskan Club Crosswalk V1

The official full asset and provider 2026 scope yield 16 name-and-competition
candidate pairs. They are deliberately **not** reviewed mappings: the current
W2 canonical team ID is unavailable for all 16, and no reviewer/timestamp is
recorded. Therefore the only valid state is `PLAYER_IDENTITY_REVIEW_PENDING`.

This package may be used as the review queue. It must not drive staging writes,
player matching, lineup value calculations, or numeric model adjustments.
