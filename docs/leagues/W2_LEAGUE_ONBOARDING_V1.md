# W2 League Onboarding V1

Each league profile defines competition mapping, country, timezone, season naming
policy, expected team count, promotion/relegation policy, market scope, data
sources, model scope, calibration scope, and readiness requirements.

The 15 onboarding checklist items are:

registration, season, teams, promotion_relegation, historical_data,
odds_coverage, market_backtest, independent_model, calibration,
strategy_validation, shadow, production, monitoring, archive, rollover.

When Gate 4 is blocked:

- strategy validation = `BLOCKED_GATE4`
- shadow = `BLOCKED_GATE4`
- production = `DISABLED`

正式推荐尚未启用。
